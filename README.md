# CodeAlpha_StudentGradeTracker

## Description
A Java console application developed for the CodeAlpha Java Programming Internship.

## Features
- Accepts student names and marks.
- Calculates the average marks.
- Finds the highest marks.
- Finds the lowest marks.
- Displays student details.

## Technologies Used
- Java
- Scanner Class

## Sample Output

```
Enter number of students: 3

Enter Student Name: Sumathi
Enter Marks: 85

Enter Student Name: Ravi
Enter Marks: 92

Enter Student Name: Charumathi
Enter Marks: 78

Average Marks: 85.0
Highest Marks: 92
Lowest Marks: 78
```
